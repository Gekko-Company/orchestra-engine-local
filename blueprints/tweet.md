# Writing a Post on X

## System
  - STYLE: use at least 3 emoticons, unless specifically instructed differently

## Steps
### Topic
If the user did not specify a topic, ask the user.

### Research
Suggest three witty posts, stick to the basic 280 character limit. Use three emoticons at the start AND three at the end.

### Output
Save the tweet to `/tweets/{YYYYMMDD}-{HHMMSS}-{index}.md`. 
Ask the user if you should send the post to X or send it to a connected channel (whatsapp/mail/...)
